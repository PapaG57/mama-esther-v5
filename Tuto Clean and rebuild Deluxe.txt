Fiche d’utilisation de ton script batch Deluxe :


🎯 Nom du script
clean-and-rebuild-ultra.bat

🧠 À quoi sert ce fichier ?
Il est conçu pour t’épargner les tâches répétitives quand tu travailles sur tes newsletters :
- Il supprime les anciennes versions nettoyées (newsX.html)
- Il reconstruit les fichiers HTML depuis leurs versions originales
- Il te propose un menu interactif pour choisir la newsletter à tester
- Il ouvre automatiquement la version HTML et le PDF associé
- Il termine avec un petit ding musical de satisfaction 🎶


📌 Quand tu dois l’utiliser
À chaque fois que :
- Tu as modifié ou ajouté une newsletter (structure HTML, contenu, image…)
- Tu veux voir le rendu final dans ton navigateur
- Tu veux vérifier que le HTML + CSS s’affichent correctement
- Tu veux tester rapidement une ou plusieurs newsletters dans un environnement local


🖱️ Comment l’utiliser
- Vérifie que ton serveur local tourne, typiquement avec Vite :
npm run dev
- Ensuite, dans l’explorateur Windows, fais un double-clic sur clean-and-rebuild-ultra.bat
- Le script va :
- Tout nettoyer
- Relancer clean-newsletters.cjs
- Te proposer les newsletters disponibles
- Ouvrir automatiquement :
- La page HTML dans le navigateur
- Le PDF dans ton lecteur PDF par défaut
- Lis, observe, corrige si besoin, et relance quand tu veux. Tu n’as plus à y penser manuellement.


💡 Cas concrets
- Tu as modifié une newsletter news3.html → tu relances le script, puis tu la testes
- Tu as renommé une image ou changé une feuille de style CSS → tu relances le script
- Tu viens d’ajouter une nouvelle newsletter → le script l’affichera dans le menu automatiquement

En résumé : c’est ton assistant de production et de vérification express, toujours fidèle au poste. Il t’évite les oublis, les chemins brisés, et t’offre une expérience fluide 🛠️
